# L0r3nA
 Herramienta de administracion remota escrita en C++

### Funciones
- Shell Inversa
- Administrador de archivos
- Administrador de procesos
- Keylogger
- Captura de camara
- Captura de microfono
- Escritorio Remoto (solo vista)
